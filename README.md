# SaaS Frontend Project

## Features
- Landing page (Figma design mock)
- Login & Register (mock auth)
- Dashboard with:
  - Home (summary cards, mock chart, mock users)
  - Features
  - Pricing
  - Settings (profile form)
- Logout works
- Fully offline, no API calls

## Tech Stack
- Next.js
- Tailwind CSS
- React (functional components)
- LocalStorage for mock auth
